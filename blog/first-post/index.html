Good question. I will maybe figure that out as I go.

I like Rudolf

One of the big motivations for playing this game for me is building Rudolf. Even for CMs where I don't run her I will still try to build her.

I like learning

I'm not the type of player to chase the meta or go for a 90% win rate every cm but I do like understanding how stuff works. This game has a lot to it. Race mechanics, banners, sparks, scenario mechanics. There is a lot. Learning about the game is at times also difficult because everything is so meta focused.

The general idea of this place is to keep notes for myself so that I can go back and figure out what I can improve on next. It's also just a thought dump I can come back to while waiting for my TP to recharge.
